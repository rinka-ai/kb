# Meta: About Conversions API

## TL;DR

Conversions API is Meta's first-party/server-side data bridge. It helps advertisers send conversion events to Meta for better measurement and optimization when browser-side signals are incomplete.

## Key Claims

- Conversions API can help advertisers better measure ad performance and attribution across the customer journey.
- The API can support both online and offline business outcomes.
- Meta connects this data-quality layer to optimization, attribution, and lift-style measurement.
- Use of Conversions API remains subject to Meta's business tools terms, off-Facebook activity controls, and data-use restrictions.

## Important Details

- Conversions API complements, rather than merely replaces, browser pixel events.
- The strategic role is signal restoration: feeding better conversion events to the delivery system can improve optimization quality.
- Data quality should include deduplication, event matching, consent handling, and event hygiene.
- Offline conversion uploads can matter for businesses where the real purchase or qualified lead happens after the click.
