## TL;DR

A practical modern deep learning text that maps the neural-network stack from supervised learning and losses through optimization, architecture families, generative models, transformers, and reinforcement learning.

## Key Claims

- Deep learning is best understood as function approximation plus learned representation, trained through loss functions and optimization.
- Architecture choices encode useful inductive biases, including composition, convolution, attention, residual connections, graph structure, and generative latent structure.
- Training behavior depends on losses, gradients, initialization, normalization, regularization, and optimization dynamics, not only model size.
- Modern deep learning includes discriminative, generative, sequence, graph, and RL-adjacent models, so a KB concept should not reduce it to feed-forward networks.
