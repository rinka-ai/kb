## TL;DR

MEM1 treats memory management as a learned behavior, training agents to continuously consolidate useful information into a compact state so long-horizon interactions do not explode in token cost.

## Key Claims

- Long-horizon agent performance depends on memory efficiency, not just raw context length.
- Agents can learn to consolidate memory end-to-end through reinforcement learning.
- Strong pruning is viable when the model learns what to preserve internally.

## Important Details

- The note describes a compact shared internal state that becomes the only retained memory.
- Earlier turns, observations, and tool outputs are pruned after consolidation.
- PPO-based reinforcement learning is used to align consolidation behavior with task success.
- The note highlights competitive results on multi-objective QA and WebShop-style environments.
