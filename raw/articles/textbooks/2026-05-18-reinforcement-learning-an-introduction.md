## TL;DR

The core RL reference for understanding agents as learners that interact with environments over time, optimize returns, update value estimates, and manage exploration.

## Key Claims

- RL differs from supervised learning because the learner acts, receives delayed rewards, and changes future data through its policy.
- MDPs, value functions, Bellman equations, and policies provide the basic mathematical language for sequential learning.
- Bandits isolate the exploration-exploitation problem before full sequential state dynamics are introduced.
- Temporal-difference learning bridges Monte Carlo returns and dynamic programming backups.
