# Textbooks

## Overview

This collection contains compact source notes for the official AI/ML textbook corpus downloaded into `/Users/josemanuelcerqueira/Desktop/mit-ai-books/`. The notes deliberately avoid copying full book text; they point to the local official PDFs/HTML mirrors and synthesize the parts that improve the wiki.

## Source Notes

- [[2026-05-18-foundations-of-machine-learning]] — Foundations of Machine Learning anchors the KB in statistical learning theory, generalization, PAC learning, Rademacher complexity, VC dimension, model selection, regularization, kernels, and online learning.
- [[2026-05-18-understanding-deep-learning]] — Understanding Deep Learning gives a modern, architecture-level map of neural networks, losses, optimization, convolution, attention, residual networks, generative models, transformers, and RL basics.
- [[2026-05-18-machine-learning-systems-vol1]] — Machine Learning Systems Vol. 1 reframes ML as an engineering lifecycle involving data, training, deployment, efficiency, monitoring, and return on compute.
- [[2026-05-18-machine-learning-systems-vol2]] — Machine Learning Systems Vol. 2 focuses on scale: distributed infrastructure, fleet dynamics, communication bottlenecks, scaling laws, governance, reliability, and sustainability.
- [[2026-05-18-algorithms-for-decision-making]] — Algorithms for Decision Making provides the core algorithms for probabilistic reasoning, sequential decision problems, model uncertainty, state uncertainty, and multiagent decision-making.
- [[2026-05-18-deep-learning-goodfellow-bengio-courville]] — Deep Learning is the canonical broad reference for representation learning, neural-network training, optimization, regularization, convolutional/recurrent networks, graphical models, inference, and generative models.
- [[2026-05-18-reinforcement-learning-an-introduction]] — Reinforcement Learning: An Introduction is the canonical source for bandits, MDPs, dynamic programming, Monte Carlo methods, temporal-difference learning, eligibility traces, planning, function approximation, and policy gradients.
- [[2026-05-18-distributional-reinforcement-learning]] — Distributional Reinforcement Learning shifts RL from expected returns to return distributions, covering distributional Bellman equations, metrics, dynamic programming, incremental algorithms, control, statistical functionals, and deep RL applications.
- [[2026-05-18-multi-agent-reinforcement-learning]] — Multi-Agent Reinforcement Learning grounds multi-agent systems in games, stochastic games, partial observability, communication, coordination, centralized training/decentralized execution, and learning dynamics.
- [[2026-05-18-algorithms-for-optimization-first-edition]] — Algorithms for Optimization gives algorithmic foundations for gradients, bracketing, descent, second-order methods, stochastic methods, constrained optimization, and surrogate or Bayesian optimization.
- [[2026-05-18-algorithms-for-optimization-second-preview]] — The second-edition preview updates the optimization coverage with modern gradient, automatic differentiation, stochastic approximation, local search, and global optimization material.
- [[2026-05-18-algorithms-for-validation]] — Algorithms for Validation gives the KB a system-assurance vocabulary: modeling, property specification, stochastic metrics, temporal logic, reachability, falsification, optimization-based testing, and validation evidence.
- [[2026-05-18-decision-making-under-uncertainty]] — Decision Making Under Uncertainty is a compact source on probabilistic reasoning, sequential decisions, model uncertainty, state uncertainty, and planning under uncertainty.
- [[2026-05-18-fairness-and-machine-learning]] — Fairness and Machine Learning reframes fairness as a socio-technical loop involving measurement, legitimacy, classification criteria, causality, feedback loops, recourse, legal context, and broader justice.
- [[2026-05-18-probabilistic-machine-learning-introduction]] — Probabilistic Machine Learning: An Introduction gives a broad probabilistic foundation for supervised, unsupervised, and reinforcement learning, with probability, generative models, Bayesian inference, and neural methods.
- [[2026-05-18-probabilistic-machine-learning-advanced-topics]] — Probabilistic Machine Learning: Advanced Topics deepens the KB around advanced probability, inference, latent-variable models, Bayesian methods, causality, deep generative models, sequential decision-making, and modern ML systems topics.
