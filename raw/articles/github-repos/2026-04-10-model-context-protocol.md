## TL;DR

The official MCP repository is the KB's primary source of truth for the protocol layer that increasingly mediates tool use and external context access in agent systems.

## Key Claims

- The repository is the canonical source for the MCP specification, schema, and documentation.
- MCP is presented as a protocol surface for integrating models with tools and external context.
- The schema is defined in TypeScript first and also published as JSON Schema for broader compatibility.
- The repo points to official documentation on `modelcontextprotocol.io`.

## Important Details

- The README frames the repo as an entry point for people newly approaching MCP.
- The source text explicitly separates specification, schema, and official documentation as distinct artifacts.
