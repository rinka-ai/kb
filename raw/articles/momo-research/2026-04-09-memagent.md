## TL;DR

MemAgent turns long-context processing into an internal memory-update problem, using reinforcement learning to maintain a fixed-length memory that scales linearly even when input grows dramatically.

## Key Claims

- Fixed-length memory can outperform brute-force long-context scaling.
- Reinforcement learning can teach the model what to retain from arbitrarily long inputs.
- Long-context competence can be extrapolated far beyond training window size when memory updates are learned well.

## Important Details

- The method uses an overwrite strategy to process long text segment by segment.
- Memory is represented as ordinary tokens inside the model context.
- The note reports strong extrapolation behavior up to multi-million-token settings on QA-style tasks.
- The training recipe uses a DAPO-derived reinforcement learning approach.
