## TL;DR

An advanced RL text arguing that learning the distribution of returns, not only its expectation, gives a richer object for theory and algorithms.

## Key Claims

- The value of a policy can be represented as a return distribution, not only an expected scalar return.
- Distributional Bellman equations and operators provide the core language for return-distribution learning.
- Probability representations, metrics, and projections matter because algorithms must approximate distributions.
- Distributional methods connect tabular theory, function approximation, control, and deep RL.
